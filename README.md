# awatake ホームページ

「79歳、AIとつくる毎日。」— awatake（1946年生まれ）の公式ホームページです。

公開URL: https://m3-kawasaki.github.io/homepage1/

## 内容

- **ストーリー** — 精密機器メーカーの技術者から中小企業コンサルタント、そしてAIプログラミングへ
- **プロフィール** — 経歴のタイムライン
- **作品ギャラリー** — AIとの共同作業（バイブコーディング）でつくったゲーム・Webアプリ 8作品
- **活動** — 画像処理同好会、シニアテック・ラボ
- **相談できること** — 企業経営・マーケティング・技術課題のコンサルティング
- **ブログ** — 健康・デジタル・コミュニティの記事

## 技術

- HTML / CSS / JavaScript（フレームワークなし）
- Google Fonts: Noto Sans JP + Shippori Mincho
- GitHub Pages でホスティング（main ブランチ直下）
